# 🤖 AI News Bot — Telegram-бот для ежедневных новостей по нейросетям

Автоматический бот, который каждый вечер собирает свежие новости по AI, нейросетям и агентам из открытых RSS-лент и отправляет сводку в Telegram.

---

## 📋 Что умеет бот

- Собирает новости из 8+ источников (VentureBeat, ArXiv, MIT Tech Review, Hugging Face и др.)
- Автоматическиcategorizирyет по темам: AI-агенты, нейросети, генеративный AI, инструменты
- Отправляет красивую сводку в Telegram каждый день в 18:00
- Работает бесплатно (без платных API)

---

## 🚀 Быстрый старт

### Шаг 1 — Получите токены

**Telegram Bot Token:**
1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Напишите `/newbot`
3. Следуйте инструкциям, сохраните токен

**Telegram Chat ID:**
1. Откройте [@userinfobot](https://t.me/userinfobot) в Telegram
2. Скопируйте ваш Chat ID (число вида `123456789`)

### Шаг 2 — Настройте config.py

Откройте `config.py` и вставьте ваши токены:

```python
TELEGRAM_BOT_TOKEN = "7123456789:AAHdq9d..."  # Ваш токен от BotFather
TELEGRAM_CHAT_ID = "123456789"                 # Ваш Chat ID
```

### Шаг 3 — Установите зависимости

```bash
cd ai_news_bot
pip install -r requirements.txt
```

### Шаг 4 — Запустите

**Тестовый запуск (отправляет новости сейчас):**
```bash
cd ai_news_bot
python -c "from bot import test_send; test_send()"
```

**Полный запуск (планировщик):**
```bash
cd ai_news_bot
python bot.py
```

Бот запустится и будет работать в фоне, отправляя новости каждый день в 18:00.

## ❓ FAQ

**Бот не отправляет сообщения?**
- Проверьте токены в `config.py`
- Проверьте, что бот добавлен в чат (для групп нужен Group ID)

**Хотите получать новости в группу?**
- Добавьте бота в группу
- Напишите `/start` в группе
- Используйте [@userinfobot](https://t.me/userinfobot) для получения Group ID (отрицательное число)

**Как остановить бота?**
- Просто закройте терминал или нажмите `Ctrl+C`

Бот полностью бесплатен и использует только открытые RSS-ленты — без платных подписок и API-ключей.
